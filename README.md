# Radar da Copa da Geografia 2026

Radar interativo educacional da **Copa da Geografia 2026**, criado pelo **Prof. Legeo (Leandro)**, professor de geografia.

É uma ferramenta web autocontida (um único arquivo `index.html`), publicada via **GitHub Pages** para ser embutida no site [legeo.com.br](https://legeo.com.br).

## Como usar

Acesse a versão publicada (GitHub Pages) ou abra o arquivo `index.html` diretamente no navegador.

## Atualizações

Para publicar uma nova versão, basta substituir o `index.html` pelo arquivo atualizado e enviar as mudanças (`git commit` + `git push`).
